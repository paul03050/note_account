# LLM_wiki — AGENTS.md

Obsidian vault for organizing and synthesizing knowledge about LLMs.

## Structure

- `raw/01-articles/` — raw ingested articles/source material
- `wiki/` — curated knowledge
  - `concepts/` — concept notes
  - `entities/` — entity notes (people, organizations, models, etc.)
  - `sources/` — source references
  - `syntheses/` — synthesized/integrated notes
  - `index.md` — vault homepage
  - `log.md` — activity log
- `assets/` — attachments (images, etc.)
- `.obsidian/` — local Obsidian config (do not edit)

## Workflow

- All content is Markdown. Use `[[wiki/internal links]]` for cross-references.
- Place new notes in the appropriate subdirectory (`concepts/`, `entities/`, `sources/`, `syntheses/`).
- Link attachments from `assets/`.
- Append new entries to `log.md`.
- Obsidian tools available: read, write, append, search, list, delete.
- No build/test/lint steps — just content creation.
